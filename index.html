Phyllotaxis Mover

Welcome to the Phyollotaxis mover. This is a project themed around creating phyllotaxis and moving them around to create interesting images. To create another phyllotaxis click on the screen adjust the controls to custimize your phyllotaxis and its movement around the canvas.

Instructions:

  • Click on the Canvas to create more phyllotaxis
  • Use controls on left to custimize phyllotaxis
  • Press play to start movement and use the movement dropbox to change move type
  • Use the checkbox to turn trails on or off and use the slider to control its length
  • Use the reset button to rest the canvas
  • Use the 'print' button to save the image on another page
  • Change the shape composing the phyllotaxis with the shape dropdown
